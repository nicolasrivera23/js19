Ejercicio 2 Clase 9

Desarrollá un programa que reciba nombre y edad de una persona, los combine en una frase
y convierta la edad de string a número para verificar si es mayor de edad.

Tips clave:
Validación edad: Comprobá que la edad ingresada sea un n° válido usando isNaN()
Concatenación: Utilizá el operador + para unir texto y personalizar el mensaje.
Conversión de tipos: Transformá la edad de string a número con parseInt() o Number() para realizar comparaciones.
Mensajes claros: Mostrá mensajes comprensibles en la consola para verificar el funcionamiento del programa.
Pruebas variadas: Probá el programa con nombres y edades diferentes, incluyendo casos como menores de edad o valores cercanos a 18, para asegurar la robustez de la lógica.
